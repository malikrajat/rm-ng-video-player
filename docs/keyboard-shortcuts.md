# 🎹 Keyboard Shortcuts

Boost your productivity and accessibility with a comprehensive set of keyboard shortcuts. Control every aspect of the playback without ever touching your mouse.

| Shortcut | Action |
| --- | --- |
| **Space** | Toggle Play / Pause |
| **K** | Toggle Play / Pause |
| **J** | Skip backward 10 seconds |
| **L** | Skip forward 10 seconds |
| **M** | Toggle Mute/Unmute |
| **F** | Toggle Fullscreen Mode |
| **C** | Toggle Closed Captions |
| **I** | Toggle Picture-in-Picture |
| **Left Arrow** | Seek backward 5 seconds |
| **Right Arrow** | Seek forward 5 seconds |
| **Up Arrow** | Increase volume by 10% |
| **Down Arrow** | Decrease volume by 10% |
| **0 - 9** | Jump to percentage of video (e.g., 5 jumps to 50%) |

## Why Keyboard Accessibility?

RM Angular Video Player follows WCAG guidelines to ensure that all users, including those with visual or motor impairments, can fully enjoy the media experience. Every button in the UI is also focusable and accessible via standard tab navigation.
