# Browser & Platform Support

RM Angular Video Player is tested across all modern browsers and major operating systems to ensure a consistent experience.

## Compatibility Matrix

### Browsers

| Browser | Version | Status | Notes |
| --- | --- | --- | --- |
| **Google Chrome** | 90+ | Supported | Full PiP and WebCodecs support. |
| **Mozilla Firefox** | 88+ | Supported | Optimized for Gecko rendering. |
| **Apple Safari** | 14+ | Supported | Native AirPlay and PiP support. |
| **Microsoft Edge** | 90+ | Supported | Full Chromium compatibility. |
| **Mobile Safari** | 14+ | Supported | Specialized iOS touch controls. |
| **Chrome Mobile** | 90+ | Supported | Optimized for Android performance. |

### Angular Support

| Version | Status | Notes |
| --- | --- | --- |
| **Angular 21** | **Recommended** | Fully leverages latest Signal APIs. |
| **Angular 20** | Supported | Compatible through backward compatibility layers. |
| **Angular 19** | ⚠Supported | May require additional polyfills for Signals. |

## Feature Support by Browser

- **Picture-in-Picture**: Supported on Chrome, Edge, and Safari.
- **Native Fullscreen**: Supported on all desktop browsers; specialized handling for iOS Safari.
- **WebVTT Captions**: Universally supported on all modern browsers.
- **4K Playback**: Dependent on hardware acceleration and browser codecs (recommended: Chrome/Edge).
