# Detailed Features

RM Angular Video Player is packed with features designed to provide a premium user experience and a robust developer foundation.

## Design Excellence

### Apple Glassmorphism
The player features a modern UI inspired by Apple's design language. Using advanced CSS backdrop filters and smooth transitions, it creates an immersive, high-end feel.

### YouTube-Style Controls
A familiar and intuitive control bar that mimics the YouTube experience. Controls automatically hide during playback and reappear on user interaction.

## Technical Architecture

### Signal-Based Reactivity
Built from the ground up using Angular 21 Signals. This ensures lightning-fast updates, minimal change detection cycles, and a reactive state management system that is easy to reason about.

### Standalone & Tree-Shakable
The library is fully modular. Being a standalone component, it reduces boilerplate and allows for optimal tree-shaking, keeping your final bundle size small.

## Playback Features

### Multiple Quality Sources
Provide multiple sources for the same video. The player includes a built-in quality switcher in the settings menu, allowing users to choose the best experience for their connection.

### Closed Captions & Subtitles
Full support for WebVTT (`.vtt`) files. Users can toggle captions on and off directly from the control bar.

### Picture-in-Picture (PiP)
Native browser PiP support allows users to watch videos in a floating window while multitasking across different tabs or applications.

### Sleep Timer
An innovative feature for relaxing or focused environments. Set a timer (15, 30, 60, 90, or 120 minutes) to automatically stop playback.

## Mobile Optimization
- **Touch-Friendly Controls**: Larger hit areas for all buttons.
- **Gesture Support**: Designed with touch interactions in mind.
- **Landscape Handling**: Optimized layout for mobile full-screen viewing.
